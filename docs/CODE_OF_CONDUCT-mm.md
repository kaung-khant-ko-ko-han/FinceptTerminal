
# အကျင့်စာရိတ္တ ကျင့်ဝတ်

## ကျွန်ုပ်တို့၏ ကတိကဝတ်

အသက်၊ ကိုယ်ခန္ဓာအရွယ်အစား၊ မသန်စွမ်းမှု၊ လူမျိုး၊ ကျားမခွဲခြားမှု၊ အတွေ့အကြုံအဆင့်၊ ပညာရေး၊ လူမှုစီးပွားအဆင့်အတန်း၊ နိုင်ငံသားဖြစ်မှု၊ ကိုယ်ပိုင်အသွင်အပြင်၊ လူမျိုးရေး၊ ဘာသာ သို့မဟုတ် လိင်စိတ်ခံယူမှု မခွဲခြားဘဲ Fincept Terminal အသိုင်းအဝိုင်းတွင် ပါဝင်သူတိုင်းအတွက် အနှောက်အယှက်ကင်းသော အတွေ့အကြုံဖြစ်စေရန် ကျွန်ုပ်တို့ ကတိပြုပါသည်။

ပွင့်လင်းပြီး၊ ကြိုဆိုတတ်ကာ၊ ကွဲပြားမှုရှိပြီး၊ လူတိုင်းပါဝင်နိုင်ကာ၊ ကျန်းမာသော အသိုင်းအဝိုင်းကို အထောက်အကူပြုသည့် နည်းလမ်းများဖြင့် လုပ်ဆောင်ရန် ကျွန်ုပ်တို့ ကတိပြုပါသည်။

---

## ကျွန်ုပ်တို့၏ စံနှုန်းများ

### ကောင်းမွန်သော အပြုအမူ

- စာနာမှုနှင့် ကြင်နာမှုကို ပြသခြင်း
- ကွဲပြားသော ရှုထောင့်များကို ရိုသေလေးစားခြင်း
- အပြုသဘောဆောင်သော တုံ့ပြန်ချက်ကို ပေးခြင်းနှင့် လက်ခံခြင်း
- အမှားများအတွက် တာဝန်ယူခြင်း
- တစ်ဦးချင်းအကျိုးထက် အသိုင်းအဝိုင်းကို အာရုံစိုက်ခြင်း
- အခြားသူများ သင်ယူကြီးထွားရန် ကူညီခြင်း

### လက်မခံနိုင်သော အပြုအမူ

- လိင်ပိုင်းဆိုင်ရာ ဘာသာစကား၊ ပုံရိပ်များ၊ သို့မဟုတ် မလိုလားအပ်သော တိုးဝင်ခြင်း
- အနှောက်အယှက်ပေးခြင်း (trolling)၊ စော်ကားခြင်း၊ အထင်အမြင်သေးသော မှတ်ချက်များ၊ သို့မဟုတ် ပုဂ္ဂိုလ်ရေးတိုက်ခိုက်ခြင်း
- အများသူငှာ သို့မဟုတ် သီးသန့် နှောင့်ယှက်ခြင်း
- ခွင့်ပြုချက်မရှိဘဲ အခြားသူများ၏ ကိုယ်ရေးအချက်အလက်များကို ထုတ်ပြန်ခြင်း
- လူမှုကွန်ရက်နှောင့်ယှက်ခြင်း (spam) သို့မဟုတ် အလွန်အကျွံ ကိုယ်ကိုတိုးမြှင့်ခြင်း
- ပရော်ဖက်ရှင်နယ် ပတ်ဝန်းကျင်တွင် မသင့်လျော်သော အခြားအပြုအမူ

---

## ငွေကြေးနည်းပညာ ကျင့်ဝတ်

ဤငွေကြေးနည်းပညာစီမံကိန်း၏ သဘောသဘာဝအရ-

**လုံခြုံရေးနှင့် ဘေးကင်းရေး**
- အသုံးပြုသူ၏ ငွေကြေးဆိုင်ရာ ဘေးကင်းမှုကို ဦးနှောက်ထားသည့် ဒီဇိုင်း
- ဘယ်သောအခါမှ ရည်ရွယ်ချက်ရှိရှိ အားနည်းချက်များကို မထည့်သွင်းရန်
- တိကျသော ငွေကြေးဆိုင်ရာ တွက်ချက်မှုများကို ထိန်းသိမ်းရန်
- ငွေကြေးကိုယ်ရေးကိုယ်တာ စည်းမျဉ်းများကို လေးစားရန်

**စျေးကွက် သမာဓိ**
- စျေးကွက်ခြယ်လှယ်ရေး အင်္ဂါရပ်များ မရှိစေရန်
- အကျိုးစီးပွားပဋိပက္ခကို ပွင့်လင်းစွာ ထုတ်ဖော်ခြင်း
- ငွေကြေးစည်းမျဉ်းများကို လိုက်နာခြင်း
- တာဝန်ရှိသော စျေးကွက်ဒေတာ ကိုင်တွယ်ခြင်း

---

## ပါဝင်ကူညီမှုစံနှုန်းများ

### Code ပါဝင်ကူညီသူများ

- သန့်ရှင်းပြီး မှတ်တမ်းတင်ထားသော code
- သင့်လျော်သော type hints နှင့် error handling
- Unit tests များ ထည့်သွင်းထားခြင်း
- ရှိပြီးသား ပုံစံများနှင့် သဘောတူညီချက်များကို လိုက်နာခြင်း

### ဒေတာ ပါဝင်ကူညီသူများ

- ဒေတာရင်းမြစ်၏ တိကျမှန်ကန်မှုကို အတည်ပြုခြင်း
- ဒေတာပေးသူ၏ ဝန်ဆောင်မှုစည်းကမ်းချက်များကို လေးစားခြင်း
- ဒေတာပုံစံများကို မှတ်တမ်းတင်ခြင်း
- ပျောက်နေသော ဒေတာကို သင့်လျော်စွာ ကိုင်တွယ်ခြင်း

### စာရွက်စာတမ်း ပါဝင်ကူညီသူများ

- ရှင်းလင်းပြီး အစပြုသူအတွက် အဆင်ပြေသော ဘာသာစကား
- Code နမူနာများ ထည့်သွင်းခြင်း
- အချက်အလက်များကို ခေတ်မီနေစေရန် ထားရှိခြင်း

---

## စည်းကမ်းထိန်းသိမ်းခြင်း

အသိုင်းအဝိုင်းခေါင်းဆောင်များသည် ချိုးဖောက်မှုများအတွက် သင့်လျော်သော အရေးယူမှုများ ပြုလုပ်လိမ့်မည်:

1. **ပြုပြင်ခြင်း** - အသေးစား ပြဿနာများအတွက် သီးသန့် သတိပေးခြင်း
2. **သတိပေးခြင်း** - အကျိုးဆက်များသတ်မှတ်ပေးထားသော တရားဝင်သတိပေးခြင်း
3. **ယာယီပိတ်ပင်ခြင်း** - အသိုင်းအဝိုင်းမှ ယာယီဖယ်ရှားခြင်း
4. **အမြဲတမ်းပိတ်ပင်ခြင်း** - ပြင်းထန်သောချိုးဖောက်မှုများအတွက် အမြဲတမ်းဖယ်ရှားခြင်း

---

## အစီရင်ခံခြင်း

**လက်မခံနိုင်သော အပြုအမူကို အောက်ပါသို့ အစီရင်ခံပါ:** support@fincept.in

ပါဝင်ရမည့်အရာများ:
- သင့်၏ ဆက်သွယ်ရန် အချက်အလက်
- ပါဝင်ပတ်သက်သူများ၏ အမည်များ
- ဖြစ်ရပ်၏ ဖော်ပြချက်
- ရက်စွဲနှင့် အချိန်
- အထောက်အကူပြု ပစ္စည်းများ (ဖန်သားပြင်ဓာတ်ပုံများ၊ လင့်ခ်များ)

**တုံ့ပြန်မှုအချိန်ဇယား:**
- ၄၈ နာရီအတွင်း လက်ခံကြောင်းအသိအမှတ်ပြုခြင်း
- အစီရင်ခံသူ၏ လျှို့ဝှက်မှုကို ထိန်းသိမ်းခြင်း
- တရားမျှတပြီး ပွင့်လင်းသော ဆုံးဖြတ်ချက်ချမှု လုပ်ငန်းစဉ်

ဖြစ်ရပ်၏အသေးစိတ်အချက်အလက်အပြည့်အစုံ၊ အယူခံဝင်ရခြင်းအကြောင်းရင်းနှင့် နောက်ဆက်တွဲအကြောင်းအရာ သို့မဟုတ် အထောက်အထားများနှင့်အတူ ၁၄ ရက်အတွင်း support@fincept.in သို့ အီးမေးလ်ပေးပို့၍ အယူခံဝင်နိုင်ပါသည်။
