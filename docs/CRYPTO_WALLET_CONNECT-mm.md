
# သင့်ပိုက်ဆံအိတ်ကို ချိတ်ဆက်ပါ — Crypto Center

**Crypto Center** သည် Fincept Terminal မှ သင်၏ $FNCPT အထောက်အထားကို ဖတ်ရှုသည့်နေရာဖြစ်သည်။ ၎င်းသည် သင်၏ အများပြည်သူလိပ်စာ (public address), SOL လက်ကျန်, $FNCPT လက်ကျန်၊ နှင့် တိုက်ရိုက် USD တန်ဖိုးကို ပြသသည်။ **သင်၏ private keys များသည် terminal ကို ဘယ်တော့မှ မထိပါ။**

## အထောက်အပံ့ပေးသော ပိုက်ဆံအိတ်များ

Terminal သည် browser တွင် Wallet Standard provider ကို ထုတ်ဖော်ပြသသည့် Solana ပိုက်ဆံအိတ်တိုင်းနှင့် အလုပ်လုပ်သည်-

| ပိုက်ဆံအိတ် | အခြေအနေ |
|---|---|
| Phantom | ✅ |
| Solflare | ✅ |
| Backpack | ✅ |
| Glow | ✅ |
| Coinbase Wallet (Solana mode) | ✅ `window.solana` မှတဆင့် |
| `window.solana` ကိုထုတ်ဖော်ပြသသည့် အခြား extension များ | ✅ |

Hardware ပိုက်ဆံအိတ်များ (Ledger, Trezor) သည် ပွင့်လင်းစွာ အလုပ်လုပ်သည် — ၎င်းတို့ကို Phantom သို့မဟုတ် Solflare မှတဆင့် ချိတ်ဆက်ပါ၊ ထို့နောက် terminal သည် အပိုထပ်ဆောင်း ပြင်ဆင်မှုမရှိဘဲ ရရှိလာသော အများပြည်သူသော့ကို မြင်ရသည်။

## ချိတ်ဆက်ပုံ အဆင့်မြင့်ရှင်းလင်းချက်

1. သင်သည် Crypto Center တွင် **Connect Wallet** ကို နှိပ်ပါ။
2. Terminal သည် သင်၏ ပုံမှန် browser တွင် `http://127.0.0.1:<random-port>` ၌ **တစ်ကြိမ်သုံး စာမျက်နှာ** တစ်ခုကို ဖွင့်သည်။
3. စာမျက်နှာသည် သင့်ပိုက်ဆံအိတ် extension အား ၎င်း၏ အများပြည်သူသော့ကို မျှဝေရန်နှင့် terminal မှ ထုတ်ပေးသော **တစ်ကြိမ်သုံး စိန်ခေါ်မှု မက်ဆေ့ချ် (challenge message)** ကို လက်မှတ်ထိုးရန် တောင်းဆိုသည်။
4. Terminal သည် ထိုလက်မှတ်ကို ဒေသတွင်းတွင် အတည်ပြုပြီး၊ အများပြည်သူသော့ကို (သာ) သင်၏ OS credential store တွင် သိမ်းဆည်းကာ၊ browser tab ကို ပိတ်သည်။
5. လက်ကျန်များသည် Solana RPC မှ စတင် streaming လုပ်သည်။

Browser tab သည် handshake ပြီးဆုံးပြီးနောက် မိမိကိုယ်ကို ဖျက်ဆီးပစ်သည်။ Local server သည် `127.0.0.1` ပေါ်တွင်သာ၊ random port ပေါ်တွင်သာ၊ handshake ၏ကြာချိန်အတွက်သာ (အများဆုံး ၂ မိနစ်) နားထောင်ပြီး၊ လတ်ဆတ်သော single-use token ကိုဆောင်သော တောင်းဆိုချက်များကိုသာ လက်ခံသည်။

## အဆင့်ဆင့် (Phantom)

1. Chrome, Brave, Edge, သို့မဟုတ် Firefox တွင် [Phantom extension](https://phantom.app/) ကို တပ်ဆင်ပါ။
2. Fincept Terminal တွင်: **Navigate → Crypto → Crypto Center** သို့ သွားပါ။
3. **Connect Wallet** ကို နှိပ်ပါ။
4. Browser tab အသစ်တစ်ခု ပွင့်လာသည်။ **Phantom** ဟု အမည်တပ်ထားသော အတန်းကို နှိပ်ပါ။
5. Phantom က သင့်အား ချိတ်ဆက်မှုကို ခွင့်ပြုရန် တောင်းဆိုသည် — မက်ဆေ့ချ်ကို ပြန်လည်သုံးသပ်ပြီး **Connect** ကို နှိပ်ပါ။
6. Phantom က ဒုတိယတောင်းဆိုချက်တစ်ခုကို ပြသသည်: *လက်မှတ်ထိုးထားသော မက်ဆေ့ချ် (signed message)* တောင်းဆိုချက်။ ၎င်းသည် `Fincept Terminal wallet-connect challenge.` ဖြင့် စတင်ကြောင်း အတည်ပြုပြီး **Sign** ကို နှိပ်ပါ။
7. Tab ပိတ်သွားသည်။ Terminal သည် သင်၏ အများပြည်သူသော့နှင့် လက်ကျန်များကို ပြသသည်။

Solflare flow သည် တူညီသည် — အစား Solflare အတန်းကို ရွေးပါ။

## အဆင့်ဆင့် (အခြား extension များ)

စာရင်းတွင် သင့်ပိုက်ဆံအိတ်ကို မမြင်ပါက၊ **Other Solana wallet (window.solana)** ကို နှိပ်ပါ။ စာရင်းမပါသော ပိုက်ဆံအိတ်အများစုသည် ၎င်း hook အောက်တွင် မိမိကိုယ်ကို ထိုးသွင်းကြသည်။

## အဆင့်ဆင့် (hardware ပိုက်ဆံအိတ်)

Terminal သည် Ledger သို့မဟုတ် Trezor နှင့် တိုက်ရိုက် **စကားမပြောပါ**။ ယင်းအစား:

1. သင်၏ hardware ပိုက်ဆံအိတ်ကို **Phantom** သို့မဟုတ် **Solflare** နှင့် တွဲဖက်ပါ (ထိုပိုက်ဆံအိတ်၏ UI အတွင်း တစ်ကြိမ်ပြင်ဆင်ခြင်း)။
2. အထက်ပါ ပုံမှန် Phantom/Solflare ချိတ်ဆက်မှု flow ကို သုံးပါ။
3. Phantom က သင့်အား challenge လက်မှတ်ထိုးရန် တောင်းဆိုသောအခါ၊ သင်၏ hardware ကိရိယာသည် မက်ဆေ့ချ်ကို ပြသလိမ့်မည် — device ပေါ်တွင် အတည်ပြုပါ။

## ချိတ်ဆက်မှု ဖြုတ်ခြင်း

Crypto Center → **Disconnect** ကို နှိပ်ပါ။ Terminal သည်:

- သိမ်းဆည်းထားသော အများပြည်သူသော့ကို secure storage မှ ဖျက်ပစ်သည်။
- လက်ကျန်များ ရယူခြင်းကို ရပ်တန့်သည်။
- Screen ကို ဗလာအခြေအနေသို့ ပြန်လည်သတ်မှတ်သည်။

Browser/ပိုက်ဆံအိတ်ကိုယ်နှိုက်သည် ၎င်း၏ ကိုယ်ပိုင် session ကို ထိန်းသိမ်းထားသည် — ဝင်ရောက်ခွင့်ကို အပြည့်အဝ ရုပ်သိမ်းရန်၊ Phantom ၏ **Settings → Trusted Apps → Revoke** ကိုလည်း သုံးပါ။

## ကျွန်ုပ်တို့ သိမ်းဆည်းသည်များ

သင့်ဒေသခံစက်ပေါ်တွင်သာ၊ သင်၏ OS credential store (Windows Credential Manager / macOS Keychain / Linux Secret Service) တွင်:

- သင့်ပိုက်ဆံအိတ်၏ **အများပြည်သူလိပ်စာ** (base58)
- UI မှ မှန်ကန်သောအမည်ကို ပြသနိုင်ရန် တံဆိပ်တို (short label) (`phantom`, `solflare`, …)
- သင်ချိတ်ဆက်ခဲ့သည့် အချိန်တံဆိပ်

ကျွန်ုပ်တို့သည် အောက်ပါတို့ကို ဘယ်တော့မှ သိမ်းဆည်း၊ ပေးပို့၊ သို့မဟုတ် ယာယီတောင် ကိုင်မထားပါ:

- Private keys
- Seed စကားစုများ
- လက်မှတ်ထိုးထားသော အရောင်းအဝယ် blob များ
- ပိုက်ဆံအိတ် provider ထံမှ authentication token များ

## ကျွန်ုပ်တို့ ဘယ်တော့မှ မလုပ်ဆောင်သည်များ

- ကျွန်ုပ်တို့သည် မည်သည့်အရာကိုမျှ auto-sign မလုပ်ပါ။ လက်မှတ်ထိုးရန် တောင်းဆိုမှုတိုင်းသည် ပိုက်ဆံအိတ်၏ ကိုယ်ပိုင် ခွင့်ပြုချက် UI မှတဆင့် သွားပါသည်။
- ကျွန်ုပ်တို့သည် $FNCPT သို့မဟုတ် SOL ၏ ထိန်းသိမ်းစောင့်ရှောက်မှု (custody) ကို ဘယ်တော့မှ မယူပါ — သင့်ပိုက်ဆံအိတ်သည် တစ်ခုတည်းသော လက်မှတ်ထိုးသူဖြစ်သည်။
- ကျွန်ုပ်တို့သည် local connect server ကို `127.0.0.1` ထက်ကျော်လွန်၍ ဘယ်တော့မှ မထုတ်ဖော်ပါ။ သင်၏ LAN ရှိ အခြားစက်ပစ္စည်းများသည် ၎င်းကို မရောက်ရှိနိုင်ပါ။
- ကျွန်ုပ်တို့သည် wallet-connect code ကို တင်ရန် remote CDN ကို ဘယ်တော့မှ မသုံးပါ။ Connect စာမျက်နှာသည် terminal နှင့် တွဲဖက်ထုတ်ပေးထားပြီး သင့်ဒေသခံ disk မှ ဝန်ဆောင်မှုပေးသည်။

## ပြဿနာဖြေရှင်းခြင်း

**Browser tab ပွင့်သော်လည်း ပိုက်ဆံအိတ်အတန်းများတွင် "detected" မပြပါ**
သင့် extension ကို တပ်ဆင်ထားသော်လည်း `127.0.0.1` အတွက် ပိတ်ထားနိုင်သည်။ `chrome://extensions` ကိုဖွင့်ပါ၊ သင့်ပိုက်ဆံအိတ်ကိုရှာပါ၊ **Details** ကိုနှိပ်ပါ၊ **Allow on localhost / all sites** ကို ဖွင့်ထားကြောင်း သေချာပါစေ။

**"Signature verification failed"**
သင့်ပိုက်ဆံအိတ်ရှိ challenge message သည် terminal မျှော်လင့်ထားသည်နှင့် မကိုက်ညီပါ။ ဆိုလိုသည်မှာ ပိုက်ဆံအိတ်သည် တောင်းဆိုထားသည်ထက် ကွဲပြားသော message အတွက် လက်မှတ်ကို ပြန်ပေးလေ့ရှိသည်။ ချိတ်ဆက်မှုအသစ်ဖြင့် ထပ်စမ်းပါ။

**"Timed out waiting for browser callback"**
သင်သည် အတည်ပြုရန် ၂ မိနစ်ထက် ပိုကြာသွားခြင်း သို့မဟုတ် browser က `127.0.0.1` သို့ မရောက်ရှိနိုင်ခြင်းတို့ကြောင့်ဖြစ်သည်။ ဒိုင်ယာလော့ခ်တွင် **Reopen browser** ကိုနှိပ်ပါ — ရှိပြီးသား handshake session သည် ခဏတာ အကျုံးဝင်ဆဲဖြစ်သည်။

**Balances များ "—" တွင် ရပ်နေသည်**
Terminal သည် အများပြည်သူ Solana RPC သို့ ပြန်ကျသွားသည်၊ ၎င်းသည် အမြန်နှုန်းကန့်သတ်ချက်ရှိသည်။ ယုံကြည်စိတ်ချရသော လက်ကျန်ရယူခြင်းအတွက် အခမဲ့ Helius API key ကို ထည့်ပါ:
1. [helius.dev](https://www.helius.dev/) တွင် အကောင့်ဖွင့်ပါ (အခမဲ့အဆင့်သည် လုံလောက်ပါသည်)။
2. Settings → Secrets တွင်၊ `solana.helius_api_key` အောက်တွင် key ကို ထည့်ပါ။
3. Balance ကတ်ပေါ်ရှိ **Refresh** ခလုတ်ကို နှိပ်ပါ။

**$FNCPT က သုညပြသော်လည်း ကျွန်ုပ်တွင် token များရှိသည်**
သင့် token များသည် terminal အတွက် configure လုပ်ထားသော mint နှင့် ကွဲပြားခြင်းကြောင့် ဖြစ်နိုင်သည်။ လက်ရှိ mint သည် pump.fun deployment အဖြစ် hard-coded ထားသည် — `9LUqJ5aQTjQiUCL93gi33LZcscUoSBJNhVCYpPzEpump`။ သင့် token များကို အခြားနေရာတွင် mint လုပ်ထားပါက၊ issue တစ်ခုဖွင့်ပါ။

## ကိုယ်ရေးလုံခြုံမှု

Terminal သည် အောက်ပါသို့ အထွက်တောင်းဆိုချက်များ (outbound requests) ပြုလုပ်သည်-

- Solana RPC endpoint (သင်သည် key ကိုသတ်မှတ်ပါက Helius၊ မဟုတ်ပါက အများပြည်သူ mainnet endpoint) — `getBalance` နှင့် `getTokenAccountsByOwner` အတွက်။ Endpoint သည် သင်၏ အများပြည်သူသော့ကို မြင်ရသည်။
- `lite-api.jup.ag` — $FNCPT spot price အတွက်။ Endpoint သည် mint လိပ်စာ (သင့်ပိုက်ဆံအိတ် မဟုတ်) ကိုသာ မြင်ရသည်။

တောင်းဆိုချက်နှစ်ခုစလုံးကို terminal မှ တိုက်ရိုက်ပြုလုပ်သည်။ သင်၏ ပိုက်ဆံအိတ် provider (Phantom/Solflare) သည် မည်သည့်အချိန်တွင်မျှ terminal မှ ဆက်သွယ်ခြင်းမခံရပါ — ၎င်းသည် သင့် browser တွင်သာ လည်ပတ်ပြီး သင့်ပိုင်ဆိုင်မှုဖြစ်သည်။
